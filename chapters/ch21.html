Chapter 21: Advanced CSS

CSS Variables

Variables allow you to store values (like colors or sizes) and reuse them throughout the stylesheet.

:root {
  --primary-color: cornflowerblue;
  --padding-std: 20px;
}

.var-box {
  background-color: var(--primary-color);
  padding: var(--padding-std);
}

.var-override {
  --primary-color: tomato; /* Local override */
}
I use the global primary color (Blue).

I use a locally overridden primary color (Tomato).

CSS Calculation calc()

Allows you to perform calculations to determine property values, mixing different units (e.g., % and px).

.calc-box {
  /* The width is 100% of the parent minus 50px */
  width: calc(100% - 50px); 
}
My width is calc(100% - 50px).
Resize the browser to see me adapt.

CSS Counters

Automatic numbering managed by CSS, not HTML.

.counter-container {
  counter-reset: section-counter;
}

.counter-container h3 {
  counter-increment: section-counter;
}

.counter-container h3::before {
  content: "Section " counter(section-counter) ": ";
}
