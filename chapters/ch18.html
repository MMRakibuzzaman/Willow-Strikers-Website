.v-hidden { visibility: hidden; } /* Space reserved */

CSS Overflow

Handling content that is too big for its container.

.ov-visible { overflow: visible; }
.ov-hidden { overflow: hidden; }
.ov-scroll { overflow: scroll; }
Visible:
This text is too long for the box but it spills out over the edge visibly.

Hidden:
This text is too long for the box and the extra content is clipped off.
Scroll:
This text is too long but you can scroll to see the rest of it.
