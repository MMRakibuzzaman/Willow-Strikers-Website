Chapter 16: CSS Text

Text Rendering

Color & Decoration

.color-tomato {
  color: tomato;
}
.deco-line {
  text-decoration: underline;
}
.deco-through {
  text-decoration: line-through;
}

I am tomato colored.

I have an underline.

I have a line-through.

Transformation

.case-upper {
  text-transform: uppercase;
}
.case-cap {
  text-transform: capitalize;
}

this was lowercase.

this text is capitalized.

Wide Word Spacing

This paragraph has a large line-height. It creates more vertical space between lines of text, improving readability for long blocks of content.

Overflow & Handling

Whitespace & Ellipsis

/* Preserves spaces and line breaks */
.space-pre {
  white-space: pre;
}

/* Truncates text with ... */
.overflow-ellipsis {
  width: 200px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}

This text preserves spaces and breaks.

This text is too long for the box and will be cut off with dots.
