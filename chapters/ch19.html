Chapter 19: CSS Layout

CSS Position

Controls where an element is placed in the document.

.p-static { position: static; } /* Default */
.p-relative { position: relative; left: 20px; } /* Offset from normal */
.p-absolute { position: absolute; top: 0; right: 0; } /* Relative to parent */
.p-sticky { position: sticky; top: 0; } /* Sticks on scroll */
.p-fixed { position: fixed; bottom: 20px; right: 20px; } /* Relative to viewport */

CSS Float

Pushes elements to the left or right, allowing text to wrap around them.

.f-right { float: right; }
