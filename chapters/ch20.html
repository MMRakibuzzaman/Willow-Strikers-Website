Chapter 20: CSS Transitions & Animations

CSS Transitions

Changes property values smoothly (over a given duration) when triggered (e.g., on hover).

Basic Transition

Hover over the box to see it grow and change color.

.trans-basic {
  transition: width 1s, background-color 1s;
}
.trans-basic:hover {
  width: 300px;
  background-color: cornflowerblue;
}
Hover Me

Transition Delay

Hover over the box. The rotation starts after 1 second.

.trans-delay {
  transition: transform 1s;
  transition-delay: 1s;
}

Transition Timing Functions

Hover over the container below to compare speed curves.

.t-linear { transition-timing-function: linear; } /* Constant speed */
.t-ease { transition-timing-function: ease; } /* Slow start, fast, slow end */

CSS Animations

Complex animations using `@keyframes` that run automatically.

Basic Animation (Keyframes)

@keyframes move-right {
  from { transform: translateX(0); }
  to { transform: translateX(300px); }
}
.anim-basic {
  animation: move-right 3s forwards;
}
Moving

Multiple Keyframes & Infinite Loop

@keyframes color-cycle {
  0% { background: tomato; }
  50% { background: yellow; }
  100% { background: purple; }
}
.anim-multi {
  animation: color-cycle 4s infinite;
}
Colors

Direction: Alternate

Plays forward, then backward.

.anim-dir-alt {
  animation: move-right 2s infinite alternate;
}
