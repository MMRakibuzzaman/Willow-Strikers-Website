Chapter 8: HTML Advanced (Display Types)

Understanding how elements behave on the page (Block vs. Inline) is crucial for layout.

1. Block Elements

Block elements start on a new line and take up the full width available. Common examples: <div>, <p>, <h1>. 

.block-element {
  display: block;
  /* Width defaults to 100% */
}
I am a Block element (Div 1).
I am also a Block element (Div 2).
Note how we stack vertically.

2. Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. You cannot set width or height on them. Common examples: <span>, <a>, <b>. 

.inline-element {
  display: inline;
  /* Width/Height properties are ignored */
}

This is a paragraph with Inline Span 1 and Inline Span 2 inside it.

3. Inline-Block Elements

These are a hybrid. They flow horizontally like inline elements, but you can set width, height, padding, and margins on them.

.inline-block-element {
  display: inline-block;
  width: 150px;
  height: 60px;
}
