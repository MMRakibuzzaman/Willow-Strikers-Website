Chapter 14: CSS Box Model

Box Sizing Types

`box-sizing` defines how the width and height of an element are calculated. 

/* Total Width = 200px + 40px (padding) + 20px (border) = 260px */
.content-box {
  width: 200px;
  padding: 20px;
  border: 10px solid blue;
  box-sizing: content-box;
}

/* Total Width = 200px (Padding and Border are included inside) */
.border-box {
  width: 200px;
  padding: 20px;
  border: 10px solid green;
  box-sizing: border-box;
}
I am Content Box (Wider)
I am Border Box (Exact Width)

CSS Margins

Standard Margin

Margin creates space outside the border.

.margin-standard {
  margin: 50px;
}
I have 50px margin on all sides.

Auto Margin (Centering)

Setting `margin: auto` on the left and right centers a block element.

.margin-auto {
  width: 50%;
  margin: 0 auto;
}
I am centered horizontally.

Collapsing Margins

Vertical margins collapse. The distance is the larger of the two, not the sum.

.box1 { margin-bottom: 50px; }
.box2 { margin-top: 30px; }
/* The gap between them is 50px, NOT 80px */
Box 1 (Bottom Margin: 50px)
Box 2 (Top Margin: 30px)

CSS Outline

An outline is drawn outside the border. Unlike border, it does not take up space.

.outline-example {
  border: 5px solid black;
  outline: 5px dashed red;
  outline-offset: 5px;
}
I have a black border and a red outline.

CSS Borders

Border Style

.border-style {
  border: 5px double purple;
}
I have a double purple border.

Border Radius

.border-radius {
  border: 2px solid black;
  border-radius: 20px;
}
I have rounded corners.

CSS Padding

Padding creates space inside the border, around the content.

.padding-example {
  border: 1px solid black;
  padding: 50px;
}
I have 50px padding. The text is pushed away from the border.

CSS Width and Height

Defining explicit dimensions.

.dimension-fixed {
  width: 300px;
  height: 100px;
}

.dimension-percent {
  width: 50%;
  height: 50px;
}
