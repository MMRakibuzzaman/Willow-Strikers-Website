Chapter 15: CSS Colors

Color Names

CSS supports 140+ standard color names that you can use directly. 

.name {
  background-color: tomato;
  color: white;
}

.name-2 {
  background-color: slateblue;
  color: gold;
}
I am colored using 'tomato'.
I am colored using 'slateblue'.

Hex Color

Hexadecimal colors start with a `#` followed by 6 characters (0-9, A-F). 

.hex {
  background-color: #ff5733;
  color: #ffffff;
}

.hex-2 {
  background-color: #28a745;
  color: #f8f9fa;
}
I am colored using Hex #ff5733.
I am colored using Hex #28a745.

RGB Colors

RGB defines colors using Red, Green, and Blue values (0-255). RGBA adds an Alpha channel for opacity (0-1). 

.rgb {
  background-color: rgb(0, 123, 255);
  color: rgb(255, 255, 255);
}

.rgba {
  background-color: rgba(0, 123, 255, 0.5);
  color: black;
}
I am solid RGB color.
I am RGBA with 0.5 opacity (50% transparent).

Grey Colors

Grey shades are created when the Red, Green, and Blue values are equal. 

.grey-light {
  background-color: #eeeeee;
  color: #333333;
}

.grey-dark {
  background-color: rgb(60, 60, 60);
  color: rgb(200, 200, 200);
}
I am Light Grey (#eeeeee).
I am Dark Grey (rgb(60, 60, 60)).
