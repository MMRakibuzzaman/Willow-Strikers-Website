Chapter 17: CSS Backgrounds

CSS Background Properties

Background Color

.bg-color {
  background-color: tomato;
}
I have a solid tomato background.

Background Repeat

.repeat-x {
  background-repeat: repeat-x;
}
.repeat-y {
  background-repeat: repeat-y;
}
.no-repeat {
  background-repeat: no-repeat;
}
Repeat Horizontal (X)
Repeat Vertical (Y)
No Repeat

Background Position

.bg-pos {
  background-position: center;
}
The image is positioned in the center.

Background Size

/* Cover: Fills the entire area (might cut off image) */
.bg-cover {
  background-size: cover;
}

/* Contain: Shows whole image (might leave empty space) */
.bg-contain {
  background-size: contain;
}
Size: Cover
Size: Contain

Background Clip

Controls how far the background extends.

/* Extends behind the border */
.clip-border {
  background-clip: border-box;
}

/* Extends to the padding edge (stops at border) */
.clip-padding {
  background-clip: padding-box;
}

/* Restricts background to content only */
.clip-content {
  background-clip: content-box;
}
Border Box
Padding Box
Content Box

Background Attachment

Controls if the background scrolls with the page or the element.

/* Background stays fixed relative to the box */
.attach-scroll {
  background-attachment: scroll;
}

/* Background scrolls WITH the content */
.attach-local {
  background-attachment: local;
}

Scroll (Default):
Scroll me!
The background
stays put
relative to border.




Local:
Scroll me!
The background

the text.




CSS Gradient Properties

Linear Gradients

.linear-down {
  background-image: linear-gradient(blue, yellow);
}
.linear-right {
  background-image: linear-gradient(to right, blue, yellow);
}
.linear-diag {
  background-image: linear-gradient(to bottom right, blue, yellow);
}
Top to Bottom
Left to Right
Diagonal

Radial Gradients

.radial {
  background-image: radial-gradient(circle, red, yellow, green);
}
Radial (Center outwards)

Repeating Gradients

.repeat-linear {
  background-image: repeating-linear-gradient(
    45deg,
    blue,
    blue 10px,
    white 10px,
    white 20px
  );
}
Repeating Pattern
