CSS Units

Absolute Lengths

This is a paragraph with font-size: 2cm

This is a paragraph with font-size: 10mm

This is a paragraph with font-size: 1in

This is a paragraph with font-size: 50px

This is a paragraph with font-size: 50pt

Relative Lengths

em unit

Parent element with `font-size: 30px;`

This is a paragraph inside the `em-parent`.

This paragraph has `font-size: 2em`. Since its parent is 30px, its actual size is (30px * 2) = 60px. 

.em-parent {
    font-size: 30px;
}
.em-child {
    font-size: 2em; /* Calculates to 60px */
}

rem unit

This is a grandparent element (with `font-size: 20px;` for demonstration, but `rem` ignores it).

This is a parent element.

This paragraph has `font-size: 2rem`. If `html { font-size: 16px; }` (browser default), its actual size is (16px * 2) = 32px. 

/* Assuming default html { font-size: 16px; } */
.rem-child {
    font-size: 2rem; /* Calculates to 32px, relative to html's 16px */
}

Shorthand Properties

This box has border, margin, and padding set with individual properties. 
.non-shorthand-example {
    font-size: 20px;
    width: 80%;
    border-color: rgb(166, 255, 0);
    border-style: solid;
    border-top-width: 5px;
    border-right-width: 10px;
    border-bottom-width: 15px;
    border-left-width: 20px;
    margin-top: 40px;
    margin-left: 40px;
    margin-bottom: 10px;
    margin-right: 40px;
    padding-left: 10px;
    padding-right: 20px;
    padding-top: 15px;
    padding-bottom: 20px;
}

Using Shorthand Properties

This box achieves the same look using shorthand properties. 
.shorthand-example {
    font-size: 20px;
    width: 80%;
    border-color: rgb(166, 255, 0);
    border-style: solid;
    border-width: 5px 20px 15px 10px; /* Top Right Bottom Left */
    margin: 40px 40px 10px 40px;       /* Top Right Bottom Left */
    padding: 15px 10px 20px 20px;       /* Top Right Bottom Left */
}
