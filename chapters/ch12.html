Chapter 12: CSS Selectors

Universal Selector (*)

Selects all elements on the page. Use it carefully.

/* Selects every element inside the example box */
.universal-example * {
  border: 1px solid red;
}

Type Selector

Selects all elements of a specific type (e.g., `h3`, `p`, `div`).

/* Selects all 
 elements in this example */
.type-example h3 {
  color: green;
}

ID Selector (#)

Selects a single element with a specific `id`. An ID must be unique.

/* Selects only the element with id="example-id" */
#example-id {
  font-weight: bold;
  color: #d9534f;
}

This text is selected by its ID.

This text is not selected.

Class Selectors (.)

Selects all elements with a specific `class` attribute.

Single Class

/* Selects all elements with class="highlight" */
.highlight {
  background-color: #f0ad4e;
  padding: 5px;
}

This paragraph is highlighted.

Multi-Class Selector

You can chain class selectors to target elements that have *all* the classes. 

/* Selects elements with BOTH "blue" AND "important" classes */
.blue.important {
  color: blue;
  font-weight: bold;
}

This is just blue.

This is just important.

This is blue AND important.

Grouping Selector (,)

Applies the same style to multiple selectors by separating them with a comma. 

/* Applies the style to 
 AND 
 elements in this example */
.grouping-example h3,
.grouping-example p {
  border-left: 3px solid #5bc0de;
  padding-left: 10px;
}

This heading has the style.

This paragraph also has the style.

This span does not.

Combinator Selectors

Selectors that combine other selectors to show relationships.

Descendant Combinator (space)

Selects elements that are descendants (children, grandchildren, etc.) of a parent. 

/* Selects any 
 INSIDE .descendant-example */
.descendant-example p {
  background-color: #f7f7f7;
}

I am a direct child paragraph (selected).

I am a nested paragraph (also selected).

Child Combinator (>)

Selects elements that are *direct children* of a parent.

/* Selects only 
 that are DIRECT children of .child-example */
.child-example > p {
  font-style: italic;
}

I am a direct child paragraph (selected).

I am nested, so I am NOT selected.

Pseudo-Class Selectors (:)

Selects elements based on their state or position (e.g., `:hover`, `:checked`, `:disabled`). 

/* Styles the link when the mouse is over it */
.pseudo-class-example a:hover {
  background-color: yellow;
}

/* Styles the button when it is disabled */
.pseudo-class-example button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
Hover over me!

Pseudo-Element Selectors (::)

Styles a specific *part* of an element (e.g., `::first-line`, `::first-letter`). 

/* Styles the first line of the paragraph */
.pseudo-element-example p::first-line {
  font-weight: bold;
}

/* Styles the first letter of the paragraph */
.pseudo-element-example p::first-letter {
  font-size: 2em;
  color: red;
  float: left;
  margin-right: 5px;
}

This is a paragraph of text. The first line will be bold, and the very first letter will be large and red. The rest of the text will flow around it normally. This is often used for a drop-cap effect.

Attribute Selectors ([])

Selects elements based on their attributes and attribute values.

/* Selects links that open in a new tab */
.attribute-example a[target="_blank"] {
  background-color: #f0ad4e;
}

/* Selects links whose href STARTS WITH "https://" */
.attribute-example a[href^="https://"] {
  font-weight: bold;
}

/* Selects links whose href ENDS WITH ".pdf" */
.attribute-example a[href$=".pdf"] {
  text-decoration: underline;
}
