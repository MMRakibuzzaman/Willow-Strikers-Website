
# The Willow Strikers - Official Team Site & Learning Portal

## 📌 Project Overview
This project is a multi-page, responsive website developed for "The Willow Strikers" cricket team. It serves a dual purpose:
1.  **Team Portal:** Showcasing team statistics, player profiles, and contact information.
2.  **Learning Hub:** A structured curriculum ("The Book") linking to 22 chapters of web development concepts.

## 🔗 Live Demo
[Click here to view the live website](https://MMRakibuzzaman.github.io/Willow-Strikers-Website/)

## ⚙️ Tech Stack
* **HTML5:** Semantic structure (`<header>`, `<main>`, `<footer>`, `<nav>`).
* **CSS3:** Custom styling with CSS Variables (`:root`), Flexbox, and CSS Grid.
* **Design:** Mobile-First responsive design using Media Queries.

## 🚀 Key Features
* **Responsive Navigation:** A complex navigation bar with hover-triggered dropdown menus for the curriculum chapters.
* **Data Visualization:** Custom-styled HTML tables for batting and bowling statistics.
* **Interactive Forms:** A contact form with validation attributes.
* **CSS Architecture:** Uses a central `style.css` with defined color variables (Primary: `#004d00`, Secondary: `#cc3300`) for consistent branding.

## 📂 Project Structure
* `index.html` - Landing page with Hero section and team highlights.
* `team.html` - Player cards using CSS Grid layout.
* `stats.html` - Data tables with hover effects.
* `book.html` - Curriculum navigation hub.
* `/chapters` - Contains 22 individual topic modules (Selectors, Box Model, Flexbox, etc.).

## 🛠️ How to Run Locally
1.  Clone the repository.
2.  Navigate to the project folder.
3.  Double-click `index.html` to open it in your browser.
